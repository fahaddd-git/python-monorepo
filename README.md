# python mono repo sample 

## what is this?

This is a sample to implement a multi-project structure in python.  
I am using poetry to manage my dependencies.  


## project structure

```
.
├── README.md
├── libs
│   ├── lib-one
│   └── logger
├── poetry.lock
├── projects
│   ├── __init__.py
│   ├── project-one
│   └── project-two
└── pyproject.toml
```
